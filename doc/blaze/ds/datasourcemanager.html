getDataSource

public DataSource getDataSource(blaze\lang\String|string dsn, blaze\lang\String|string uid, blaze\lang\String|string pwd, array options)

Returns a DataSource to a ressource with the given configuration

Parameters:
dsn - The connection string to the datasource, begins with bdsc (blaze-data-source-connectivity) and is an URI
Structure:
bdsc::://[:Port][/DB][?UID=User][&PWD=Password][&Option=Value]...

Currently there are the following gateways, each of them have their own drivers:
  • native - Native PHP libraries(e.g. see mysql_connect())
  • pdo - All available PDO-Drivers

Options are driver specific. Option names are caseinsensitive and the order does not matter.

Example:
bdsc:pdo:mysql://localhost:3306/mydb
uid - The uid which is used to establish a connection
pwd - The password which is used to establish a connection
options - Driver specific options.
Returns:
A Datasource to the given dsn.
Throws:
Is thrown if a database error occurs.
