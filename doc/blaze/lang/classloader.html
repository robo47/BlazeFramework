Class ClassLoader

Object
    └─ClassLoader

public class ClassLoader
extends Object

This class sets up the classpath and can load classes. It includes the autoloader which is necessary to be able to use the namespaces. The class Object is already included in the index.php, so ClassLoader can extend it.
