write

public void write(blaze\lang\String|string str, integer off, integer len)

Writes the value of $str into the ressource which is used by the writer. With $off you can define a start point in $str and $len specifies the number of chars you want to write.

Parameters:
str - The value you want to write
off - The start offset in the data
len - The number of chars to write, if this value is -1 then $str is written from $off to the end of $str
Throws:
Is thrown when an IO error occurs or when the stream is already closed
